# বারোমাস — Bangla Calendar

একটি সম্পূর্ণ এবং আধুনিক বাংলা পঞ্জিকা অ্যাপ্লিকেশন। তিথি, নক্ষত্র, ঋতু, উৎসব এবং আরও অনেক কিছু এখানে পাওয়া যাবে। বারোমাস আপনার দিনের সাথী।

## Features
- Dynamic Bangla Calendar (Panjika)
- Astronomical calculations based on location
- Personal events and festival tracking
- City-specific timings
- Dark mode support

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- TanStack Query
